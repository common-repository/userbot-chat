== Description ==

Userbot helps you to reply to all messages quickly, easily and in a personalized manner. It automates frequent answers and learns from answers given by human Agents. You're never 'out of office' for anyone. You're there 24 hours a day, 7 days a week!

With the Userbot plugin, you can add the chat widget in your website in a few clicks and start to automate your conversations with AI!

Note: This plugin injects a Javascript snippet on your website frontend containing dynamic user data. Some caching solutions will cache entire pages and should not be used with this plugin. Doing so may cause conversations to be delivered to the wrong user.

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload the plugin files to the `/wp-content/plugins/userbot-chat` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. If you don't have a Userbot account, [sign up](https://userbot.ai/)
4. Login on [Userbot Dashboard](https://my.userbot.ai)
5. Go to Integrations page and retrieve the credentials
6. Use the your Wordpress Admin on Settings->Userbot screen to configure the plugin with the Userbot credentials
7. Enjoy the plugin :)
